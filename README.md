# movie-manager-python

A Python command-line CRUD application to manage a movie collection, 
built using Object-Oriented Programming (OOP) principles.

## About
This program allows users to add, view, update, and delete movies 
via a console menu. Built as part of a lab exercise at Dalarna 
University, focusing on classes, CRUD operations, and OOP design.

## Features
- Movie class with private attributes (ID, title, genre, playing time)
- Getter and setter methods for all attributes
- Full CRUD operations: Add, Present, Update, Delete
- Console menu system with named constants for menu options
- Unique ID validation to prevent duplicate entries
- Update individual fields or all fields at once
- Delete confirmation prompt
- Full input validation throughout

## How to Run
```bash
python movies.py
```

## Built With
- Python 3
- OOP — classes, private attributes, getter/setter methods
- CRUD operations via dedicated functions
- Input validation functions for clean, reusable code
